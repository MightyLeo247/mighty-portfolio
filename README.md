# 🚀 MightyLeo | Professional Portfolio

Welcome to my personal portfolio! This is a high-performance, responsive web application designed to showcase my journey as a Web Developer and Designer.

**Live Demo:** [(https://mightyleo247.github.io/mighty-portfolio/)]

---

## 🎨 Design Philosophy
This portfolio was built with a **Premium Dark-Mode** aesthetic, focusing on:
* **Glassmorphism:** Modern UI depth using backdrop filters.
* **Fluid Typography:** Using CSS `clamp()` for perfect readability on all devices.
* **Platform Agnostic:** Fully responsive from 300px mobile screens to 4K monitors.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for SEO.
* **CSS3:** Custom variables, Flexbox, Grid, and complex animations.
* **JavaScript (ES6):** Dynamic typing effects, theme switching, and intersection observers.
* **Formspree:** Professional backend-less contact form integration.

## ✨ Key Features
* **Theme Toggle:** Seamless transition between Light and Dark modes.
* **Dynamic Animations:** Scroll-triggered reveals and typing effects.
* **Asset Optimization:** Integrated professional branding (Logo & Profile).
* **CV Integration:** Direct download functionality for recruiters.
